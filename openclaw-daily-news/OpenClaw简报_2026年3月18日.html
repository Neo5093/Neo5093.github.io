⚠️ 安全与漏洞通告

高危

CVE-2026-25253: OpenClaw 远程代码执行漏洞

控制 UI 对 URL 查询字符串中 gatewayUrl 参数过度信任，允许攻击者通过恶意链接实现跨站 WebSocket 劫持，进而窃取 API 令牌、执行任意命令。CVSS 评分: 8.8。

高危

CVE-2026-25475: 路径遍历漏洞

v2026.1.30 之前版本存在路径遍历漏洞，isValidMedia() 函数未正确验证文件路径，攻击者可利用 MEDIA:/path/to/file 读取系统任意文件，包括凭据和配置文件。

警告

ClawHub 恶意插件泛滥

安全研究显示 ClawHub 平台存在恶意 Skill 插件，已发现 1,184+ 个有毒技能，攻击者通过伪造 GitHub 组件仓库实施供应链投毒，窃取用户账号和钱包信息。建议使用 SkillVetter 进行安全审核。

风险

公网暴露风险

据安全报告称，约 13.5 万+ OpenClaw 实例公网暴露，3 万+ 设备可被一键接管。85% 实例因默认绑定公网 IP (0.0.0.0:18789) 且无认证，遭黑客批量扫描。建议通过虚拟机或 Docker 容器运行，避免在主力机直接部署。
