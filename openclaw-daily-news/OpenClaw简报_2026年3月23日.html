3 安全与漏洞通告
! CVE-2026-25253 跨站 WebSocket 劫持
Control UI 模块从 URL 读取 gatewayUrl 参数未做验证，攻击者可在恶意网页中注入 JS 窃取认证 Token。CVSS 8.8，已修复。
CVE-2026-25253
! CVE-2026-25475 任意文件读取
MEDIA 路径解析函数未校验文件路径，攻击者可用目录穿越序列读取任意文件。已修复。
CVE-2026-25475
! CVE-2026-24763 Docker 沙箱命令注入
Docker 沙箱模式下 PATH 环境变量未经转义直接拼接 shell 命令，攻击者可执行任意命令。已修复。
CVE-2026-24763
! ClawHub 恶意 Skill 预警
ClawHub 平台已发现 341+ 恶意 Skills，主要伪装为加密货币工具、YouTube 助手、自动更新程序等。感染后可窃取 API 密钥、部署木马。已识别部分恶意技能：clawhub 仿冒系列、polymarket-trader、better-polymarket、rankaj、moltbook-registry 等。
! 提示词注入风险
攻击者通过网页、邮件等构造隐藏恶意指令，诱导 OpenClaw 读取后执行未授权操作（如泄露环境变量）。建议禁用自动读取网页功能。
