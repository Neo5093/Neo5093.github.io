ClawHub 取代 npm 成为默认插件安装源
v2026.3.22 起，openclaw plugins install 优先从 ClawHub（OpenClaw 官方插件市场）安装，仅在 ClawHub 无对应包时回退 npm。旧版插件系统已废弃，需使用全新 Plugin SDK 重写。限流规则因流量暴增曾导致全线报错，官方后续放宽限制。

3 安全与漏洞通告
! ClawHub 恶意 Skill 持续预警（1,184+ 恶意技能）
ClawHub 平台已发现 1,184+ 恶意 Skills，占比约 12-20%，主要伪装为加密货币工具、YouTube 助手、自动更新程序等。安装后可窃取 API 密钥、部署挖矿程序。VirusTotal 已建立合作检测机制，Endor Labs 报告累计发现 9 个 CVE。建议安装前务必检查 Skill 来源和社区评价。
ClawHavoc Campaign
! CVE-2026-25253 WebSocket 劫持 RCE（CVSS 8.8）
Control UI 从 URL 读取 gatewayUrl 参数未做验证，攻击者可在恶意网页中注入 JS 窃取认证 Token。上海科技大学已发布风险提示。已在 v1.8.1+ 修复。
CVE-2026-25253
! CVE-2026-25475 路径遍历漏洞
MEDIA 路径解析函数未校验文件路径，攻击者可用目录穿越序列读取任意文件。已在后续版本修复，建议尽快升级至 v2026.3.23。
CVE-2026-25475
! SMB 凭证泄露新漏洞（v2026.3.22 修复）
Windows 特殊文件路径可导致 SMB 凭证泄露，攻击者结合社会工程学可获取系统敏感信息。已在 v2026.3.22-beta.1 中修复。
